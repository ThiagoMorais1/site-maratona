Como Estudar?





Se você estuda para maratona de programação, automaticamente está se aprofundando em alguma linguagem de programação. Seu desafio é: Qual linguagem escolher?



Linguagens Utilizadas



Muitas linguagens são permitidas, mas as principais e mais utilizadas são:

  1. C

  2. C++

  3. Java

  4. Python

Então, se você ainda não iniciou seus estudos, escolha uma linguagem para dominar. Uma dica: os maratonistas e professores geralmente indicam o C++.




Programação Básica





Todo maratonista preparado deverá ter no mínimo conhecimento sobre os seguintes assuntos:

  1. Formatação de Saída de Dados

  2. Operadores matemáticos

  3. Condicionais

  4. Biblioteca CMATH (em C++)

  5. Laços de repetição (FOR e WHILE)

  6. Arrays Unidimensionais (vetores)

  7. Arrays Bidimensionais (Matrizes)


Estruturas de Dados





Avançando para além da parte básica destaca-se as principais estruturas de dados necessárias para a compreensão e resolução da maioria dos problemas:


Obs.: Em c++, a biblioteca STL (#include bits/stdc++.h) já contem todas as estruturas a seguir, ou seja, não necessitam que sejam implementadas pelo próprio programador.

  1. Pair

  2. Pilha (Stack)

  3. Fila (Queue) e Fila de Prioridades (Priority Queue)

  4. Lista Simples (Foward), Dupla (List) e Vector (Estilo Array)

  5. Ordenação (Bubble Sort, Quick Sort, Ordenação da STL C++)

  6. Deque (Double Ended Queue)

  7. Conjuntos (Set, Multiset, Unordered_Set e Unordered_Multiset)

  8. Dicionários (Map, Multimap, Unordered_Map, Unordered_Multimap, Hash_Set e Hash_Multiset)

  9. Hash_Map e Hash_Multimap

  10. Iteradores


Outros Assuntos Importantes





Com uma base sólida em programação básica e estrutura de dados, já é possível resolver pelo menos um problema em qualquer prova, considerando suas primeiras fases. Mas, se o competidor deseja ir além dos conhecimentos básicos, aqui estão outros assuntos de maior complexidade comumente abordados nas provas, são eles:

  1. Busca Linear e Busca Binária

  2. Recursão

  3. Backtracking

  4. Algoritmos Gulosos

  5. Expressões Lambda

  6. Programação Dinâmica

  7. Árvores

  8. Grafos


Como estudar tudo isso?





Existe diversos sites que apresentam exercícios no estilo maratona de programação. Um deles bastante utilizado no Brasil é o BeeCrowd, que possui uma gama de exercícios em português, abrangendo desde o mais básico até algoritmos muito complexos.


O Beecrowd possui também um sistema de rankeamento entre universidades além de rankings semanais, diários e várias outras features que possam funcionar como imporantes incentivadores para competições internas na plataforma.
